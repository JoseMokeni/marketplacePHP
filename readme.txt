Pour lancer le serveur, il suffit d'executer la commande suivante dans le dossier "public" :
```
php -S localhost:3000
```
Et de se rendre à l'adresse suivante : http://localhost:3000

Le compte administrateur par défaut est :
```
email : admin@admin.com
password : admin
```

Le script de création la base de données MySql se trouve dans le dossier principal du projet sous le nom de "database.sql".
